Freelancers often struggle to access credit due to irregular income and lack of formal financial records. Traditional credit scoring systems fail to evaluate such profiles effectively.
This project presents an AI-powered Credit Intelligence System that predicts a synthetic credit score using machine learning. It enhances evaluation by:
•	Accepting manual input OR bank statement uploads 
•	Using OCR (Optical Character Recognition) to extract financial data 
•	Applying feature engineering + ML model 
•	Providing loan decision + financial analysis report 
This system helps lenders make smarter decisions and empowers freelancers with financial insights.
